# Soul Site Demo

A minimal full‑stack demo site for the "Soul Site" project — starter code, build/run instructions, and contribution notes.

## Features
- Static landing pages
- Client-side interactivity (React/Vue optional)
- Simple API endpoints for demo data
- Local development and production build scripts

## Tech stack
- Node.js
- (Optional) React / Vue / Svelte
- Express / Fastify (backend)
- Vite / Webpack (bundler)

## Getting started

### Prerequisites
- Node.js (LTS)
- npm or yarn
- Git

### Install
```bash
git clone <repo-url>
cd "Soul site demo"
npm install
# or
# yarn
```

### Run (development)
```bash
npm run dev
# or
# yarn dev
```

### Build (production)
```bash
npm run build
npm start
```

## Testing
```bash
npm test
# or
# yarn test
```
